# ml_classification_brendan
Applied Machine Learning midterm project. Uses classification to identify poisonous mushrooms.  
[Link to dataset](https://archive.ics.uci.edu/dataset/73/mushroom)  
[Link to notebook](https://github.com/reedbc1/ml_classification_brendan/blob/main/classification_brendan.ipynb)  
[Link to peer review](https://github.com/reedbc1/ml_classification_brendan/blob/main/peer_review.md)  

## Setting up the virtual environment
1. Navigate to the project directory in your terminal
2. Create a virtual environment folder with "python venv .venv"
3. Enter the virtual environment with ".venv/Scripts/activate"
4. Install dependencies with "pip install -r requirements.txt"

## Running the notebook
### Option 1: Run inside VSCode  
1. Add the Jupyter extension (if not added already)
2. Open the .ipynb file titled "classification_brendan.ipynb"
3. Select kernel from virtual environment
4. Click "run all" (or run cells individually)  

### Option 2: Run with Jupyter
1. Install jupyter if you don't already have it with "pip install notebook"
2. Launch Jupyter notebook with "jupyter notebook"
3. Click on the link (something like http://localhost:8888/tree)
4. Open the .ipynb file titled "classification_brendan.ipynb"
